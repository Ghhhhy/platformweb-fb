# vue-cli3-dll-eslint

## Project setup
```
cnpm install
```

## Precompiled public files
```
cnpm run dll
```

### Compiles and hot-reloads for development
```
cnpm run serve
```

### Compiles and minifies for production
```
cnpm run build
```

### Run your unit tests
```
cnpm run test:unit
```

### Run your end-to-end tests
```
cnpm run test:e2e
```

### Lints and fixes files
```
cnpm run lint
```
