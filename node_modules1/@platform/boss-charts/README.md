# 安装

使用YARN安装
```cmd
yarn 
```

使用NPM，打包插件
```cmd
npm run lib
```

使用NPM，发布打包插件
```cmd
npm run pb
```

使用NPM，解包打包插件
```cmd
npm run unpb
```

使用NPM，运行事例工程
```cmd
npm run serve
```

使用NPM，安装打包事例工程
```cmd
npm run build:demo
```
